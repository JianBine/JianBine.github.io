vi/vim删除:一行，一个字符，单词，每行第一个字符 命令

x 删除当前光标下的字符
dw 删除光标之后的单词剩余部分
d$ 删除光标之后的该行的剩余部分
dd 删除当前行
c 功能和d相同，区别在于完成删除操作后进入INSERT MODE
cc 删除当前行，然后进入INSERT MODE

:%s/^.//g 删除每行第一个字符

Hexo部署到GitHub上面

GitHub端：
1.创一个新的repository，名字(例如:用户名为JianBine,则取名为JianBine.github.io)
2.创建完毕，打开浏览器尝试访问JianBine.github.io即可访问
3.准备完毕，开始建立本地项目
本地Hexo项目:
1、进入到Hexo目录里面，修改_config.yml
详细方法:拉到最底部，添加一下内容:
deoloy:
type: git
repo: https://github.com/JianBine/JianBine.github.io.git #注释，在仓库右侧Clone or download打开即可看到地址
branch: master
2、修改完毕，在任意地方右键选择Git Bash Here
2、安装部署所需要的插件,执行npm install hexo-deployer-git –save即可
3、结束后，执行hexo clean && hexo g && hexo deploy即可重新发布到github上面
4、打开浏览器输入自己的URL即可，例如：JianBine.github.io

Hexo博客乱码问题决解方法

1、将博客文件保存未UTF-8,用记事本打开后缀未”xxx.md”的博客文件，点击另存为，并把编码改为UTF-8
2、重新生成，部署
